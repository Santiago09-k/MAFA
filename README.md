# MAFA

El proyecto consiste en un sistema de control y monitoreo para un gabinete de almacenamiento de medicamentos. Utiliza un ESP32 para leer sensores de temperatura (DHT11), humedad, movimiento (MPU6050), luz (LDR) y proximidad (ultrasónico). Dependiendo de las condiciones internas (temperatura, humedad, apertura de puerta, movimiento no deseado), el sistema activa indicadores LED, un buzzer de alarma y ejecuta rutinas de apertura/cierre mediante servomotores. Además, implementa un ciclo de desinfección con una lámpara UV-C, envía datos a un servidor remoto y permite la interacción a través de un bot de Telegram.
